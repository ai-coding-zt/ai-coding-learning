## 热门主题

### MCP 协议

Model Context Protocol - AI 与数据源的通用连接标准

- 10+ 官方 SDK（Python, TypeScript, Go, Java...）
- 50+ 热门服务器
- 完整代码示例

[学习 MCP →](07-advanced/mcp/)

### AI 编程技能

Skills - 可重用的 AI 指令包，扩展代理能力

- 创建自定义技能
- 渐进式披露
- 跨工具复用

[学习 Skills →](07-advanced/skills/)

### 智能体编程

Agentic Coding - AI 代理的自主规划与多步执行

- Chain-of-Thought
- ReAct 模式
- 多智能体架构

[学习智能体 →](07-advanced/agentic-coding/)

### 开源工具

OpenCode 等开源 AI 编程代理

- 75+ LLM 提供商
- 完全开源
- 自托管选项

[了解更多 →](07-advanced/opencode/)
