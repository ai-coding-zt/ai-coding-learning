### 国际工具
- [GitHub Copilot](./github-copilot.md) - 最知名的 AI 编程助手
- [Cursor](./cursor.md) - AI 原生编辑器领导者

## 🎯 快速选择指南

```
预算充足 + 英文场景优先 → GitHub Copilot / Cursor
追求免费 + 功能全面 → Codeium / CodeGeeX
深度中文优化 → 通义灵码 / Trae
喜欢终端工作流 → Claude Code
企业级安全需求 → Amazon CodeWhisperer
```
