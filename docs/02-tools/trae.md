# Trae

> 字节跳动推出的 AI 原生 IDE

## 🎯 概述

Trae 是字节跳动推出的 AI 原生集成开发环境，专为中文开发者优化。它深度集成了字节跳动的大模型能力，提供从代码生成到项目管理的全套 AI 辅助功能。

**发布日期**: 2024年1月

**开发商**: 字节跳动 (ByteDance)

**定位**: 中国开发者的 AI 原生 IDE

## ⭐ 核心功能

### 1. Builder 模式

Trae 最强大的功能 - 自然语言生成完整项目：

```
输入: "创建一个电商后台管理系统，使用 React + Ant Design，
      包含商品管理、订单管理和用户管理模块"

Trae 会自动:
✓ 创建项目结构
✓ 生成路由配置
✓ 编写页面组件
✓ 生成 Mock 数据
✓ 配置样式和主题
```

### 2. Chat 模式

- **代码问答**: 询问代码相关问题
- **实时协作**: 边写代码边与 AI 对话
- **代码解释**: 选中代码获取详细解释
- **错误修复**: 自动识别并建议修复方案

### 3. 代码补全

```python
# Trae 能理解的上下文包括：
# - 当前文件内容
# - 项目依赖和配置
# - 命名规范和风格
# - 相关文件引用

def calculate_order_total(order_id):
    """计算订单总价，包括折扣和税费"""
    # Trae 会自动：
    # 1. 查询订单详情
    # 2. 计算商品总价
    # 3. 应用优惠券
    # 4. 计算税费
    # 5. 返回最终金额
```

### 4. 中文优化

- **中文注释理解**: 完美支持中文编程注释
- **中文变量名**: 支持中文命名建议
- **中文问答**: 用中文提问，用中文回答
- **中文文档**: 内置中文编程文档

## 🚀 使用技巧

### Builder 模式最佳实践

```markdown
1. 描述清晰的需求
   "创建一个 RESTful API，使用 Express.js 和 MongoDB，
    实现博客文章的增删改查功能"

2. 指定技术细节
   "使用 Mongoose 作为 ODM，JWT 进行认证，Jest 编写测试"

3. 分步骤生成
   第一步："创建项目基础和配置文件"
   第二步："编写数据模型"
   第三步："实现 API 路由"

4. 逐步审查
   每生成一个模块就检查质量
   有问题及时提出修改要求
```

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘+B | 打开 Builder |
| ⌘+L | 打开 Chat |
| Tab | 接受建议 |
| ⌘+Enter | 提交 Builder 请求 |
| ⌘+/ | 切换注释 |
