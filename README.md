## 🛠️ 涵盖的工具

### 国际工具
- **GitHub Copilot** - GitHub × OpenAI 的编程助手
- **Cursor** - 基于 VS Code 的 AI 原生编辑器
- **Claude Code** - Anthropic 的终端 AI 编程工具
- **Codeium** - 免费的 AI 编程助手
- **Amazon CodeWhisperer** - 亚马逊的编程助手

- **Trae** - 字节跳动的 AI 原生 IDE
- **通义灵码** - 阿里云的智能编码助手
- **CodeGeeX** - 智谱 AI 的代码生成模型
- **iFlyCode** - 科大讯飞智能编码助手
- **KimiCode** - 月之暗面的编程工具
- **OpenCode** - 开源 AI 编程工具
